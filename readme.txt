+---------------------+
 序列号自动生成服务器
+---------------------+

ffsnserver.exe 是一个序列号自动生成的 http 服务器，可以根据
小豚项目的 SN 编码规则自动生成序列号


使用方法
--------

运行程序：
ffsnserver.exe --port=8000

--port 用于指定 http 服务器端口号

运行成功后，通过 web 浏览器发起 http 请求可以获取 SN


以下的 url 地址用于获取 IP300B 的 SN
http://localhost:8000/DPH-IP300B

以下的 url 地址用于获取 IP350 的 SN
http://localhost:8000/DPH-IP350

服务器会在程序目录生成 log 文件，保存每天的 SN 请求日志


注意事项
--------
SN 的生成会保证自动增加，并且是唯一的，这是通过 log 文件记录已经生成的 SN
号码来保证的。因此一定不要删除生成的 log 文件，否则计数器清零，SN 会重新
分配，会有出现 SN 重号的风险。
